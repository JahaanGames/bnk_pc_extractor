16-19: some offset, same as 28-31
20-23: ?

SOME DATA (at offset [16-19] + DMAV header size [36]), 212 bytes long:
0-4: if not equal to 2, then this is subtitle data(?), with two entries total. if equal to 2, it means 28 subtitle entries FOLLOW this. 0 means no subtitles and the DMAV section is over
